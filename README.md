# 실손보험 계산기 Pro (Insurance Calculator)

1세대부터 4세대까지의 실손의료비 예상 지급액을 한눈에 비교할 수 있는 웹 애플리케이션입니다.

## 주요 기능 (Features)

*   **세대별 비교**: 1~4세대 실손보험의 예상 지급액을 실시간으로 계산하여 비교합니다.
*   **엑셀 연동**:
    *   **업로드**: 대량의 진료 내역을 엑셀로 일괄 등록할 수 있습니다.
    *   **다운로드**: 계산된 결과와 상세 내역을 엑셀 파일로 저장할 수 있습니다.
*   **상세 입력**: 급여(본인부담/공단부담/전액본인부담), 비급여(선택/이외) 등 5가지 상세 항목을 지원합니다.
*   **반응형 디자인**: PC와 모바일 환경 모두에 최적화된 최신 UI를 제공합니다.

## 사용 방법 (Usage)

1.  **영수증 추가**: 화면의 [+ 영수증 추가하기] 버튼을 눌러 개별 건을 입력하거나,
2.  **엑셀 업로드**: [엑셀 양식 다운로드]를 받아 내용을 채운 뒤 [엑셀 업로드]를 합니다.
3.  **결과 확인**: 하단의 요약표에서 세대별 지급액을 비교합니다.
4.  **결과 저장**: [결과 엑셀 다운로드] 버튼으로 계산 결과를 파일로 보관합니다.
